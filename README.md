# Behavioural Travel Modelling

This repository contains R scripts and datasets used for analyzing and modeling intercity travel behavior. The project involves estimating travel choice models based on the provided dataset. 

## 📂 Repository Structure

The repository consists of the following files:

* **`intcity.csv`**: The primary dataset containing intercity travel data used for the modeling exercises.
* **`d&e_assign3.R`**: The main R script for Assignment 3, containing the core data processing and modeling steps.
* **`Q_A.R`**: R script containing the analysis and model estimation for Question A.
* **`Q_B_M1.R`**: R script containing the analysis and estimation for Question B (Model 1).
* **`image_c7e50b.png`**: Visual output/plot generated from the R scripts (e.g., model results or data distribution).
